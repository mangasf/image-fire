# (MPWAR Performance) VirtualMachine

## How to run the VM

```
cd mpwarAnsible/vagrant
vagrant up 
```

## What has this machine?

- **Apache**, port: 80
- **Mysql**, port: 3306
- **Blackfire**
- **RabbitMQ**, port: 5672
- **RabbitMQ Dashboard**, port: 15672, user: admin, password: admin
- **Redis**, port: 6379
- **Elasticsearch**, port: 9200
- **Kibana**, port: 5601
